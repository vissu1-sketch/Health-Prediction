# 🏥 Multi-Disease Health Risk Predictor

## 📌 Project Overview
This project is a machine learning-based web application that predicts the risk of:
- Diabetes 🩸
- Heart Disease ❤️

It helps users understand their health condition early and take preventive actions.

---

## 🚀 Features
- Predicts multiple diseases
- Provides risk percentage
- Gives health suggestions
- Simple and clean user interface

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Machine Learning: Scikit-learn
- Database: CSV datasets
